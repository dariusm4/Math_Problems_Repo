# Solution

## Task 4 — Weekly Weather Observation

1. Sample space for weather on one day:

\(\Omega_1 = \{S, C, R\}\), where
- \(S\) = sunny,
- \(C\) = cloudy,
- \(R\) = rainy.

2. Sample space for two consecutive days:

\(\Omega_2 = \{(d_1, d_2) : d_1, d_2 \in \{S, C, R\}\}\).

3. Sample space for seven consecutive days:

\(\Omega_7 = \{(d_1, d_2, d_3, d_4, d_5, d_6, d_7) : d_i \in \{S, C, R\} \text{ for } i = 1,\dots,7\}\).

4. Number of elementary outcomes:

- \(|\Omega_1| = 3\)
- \(|\Omega_2| = 3 \times 3 = 9\)
- \(|\Omega_7| = 3^7 = 2187\)

5. Interpretation of an elementary outcome:

- In \(\Omega_1\): the weather state on a single day.
- In \(\Omega_2\): an ordered pair of weather states over two consecutive days.
- In \(\Omega_7\): an ordered sequence of weather states over seven consecutive days, describing the complete weekly weather pattern.
